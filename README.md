# Sellcon 프로젝트

**Sellcon**은 사용자 간 기프티콘 거래를 위한 플랫폼입니다. 이 프로젝트는 사용자들이 기프티콘을 안전하고 편리하게 거래할 수 있도록 설계되었습니다.

<br/>

## 📚 프로젝트 소개
Sellcon은 기프티콘을 판매하거나 구매하고자 하는 사용자들을 연결하는 플랫폼입니다. 사용자들은 자신의 기프티콘을 등록하고, 판매할 수 있으며, 원하는 기프티콘을 구매할 수 있습니다.

<br/>

## 🛠️ Tech Stacks
<div align="center">
  <img src="https://github.com/dlwhdska/SellconProject/assets/150890939/5fc6adb2-c81c-4c49-a0f4-6eb5135f7248" alt="Tech Stacks" width="600"/>

- **Backend:** Spring Boot, JPA
- **Frontend:** Thymeleaf, JavaScript, HTML/CSS
- **Database:** MySQL
- **Version Control:** Git & GitHub
- **Deployment:** AWS
