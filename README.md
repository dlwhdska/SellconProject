## 프로젝트 소개
SellconProject는 사용자 간에 기프티콘을 거래할 수 있는 플랫폼입니다. 이 프로젝트는 사용자들이 보다 안전하고 편리하게 기프티콘을 사고팔 수 있도록 돕습니다.
